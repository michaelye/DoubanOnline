DoubanOnline
============

模仿“豆瓣线上活动”制作的Android客户端

本来想实现所有的功能，但是做到后面后来发现豆瓣有些Advanced接口不提供，访问的时候提示need permission。
所以没有账户体系，该App只能浏览线上活动的内容。

项目已经在GooglePlay上架
